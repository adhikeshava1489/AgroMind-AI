# AgroMind-AI 🌱
AI-powered crop disease detection system using CNN, weather forecasting with LSTM, and smart agricultural risk prediction. Built with React, Flask, TensorFlow, and deep learning.

AgroMind AI is a deep learning-based smart farming platform designed to assist farmers and agricultural researchers in identifying crop diseases and predicting environmental risks. The system uses a Convolutional Neural Network (CNN) to analyze crop leaf images and detect diseases with high accuracy, while an LSTM-based weather forecasting module helps predict future weather conditions and agricultural risks. By combining computer vision and time-series forecasting, AgroMind AI aims to support smarter farming decisions, improve crop health monitoring, and reduce agricultural losses through AI-powered insights.

🚀 Features
🌿 Crop disease detection using CNN
🌦 Weather forecasting using LSTM
📊 Agricultural risk prediction
🖼 Upload leaf images for instant analysis
⚡ Real-time predictions using Flask API
🎨 Modern React frontend UI
